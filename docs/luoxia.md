# 落霞宗 · 内容规格

> **落霞世界包唯一真相源**（剧情/信息差/钩子）。引擎分层见 `architecture.md`。

---

## 0. 核心设计原则（先对齐）

1. **只定底层锚点**：开局仿真用的核心真相、人物动机、世界运转规则。无强制剧情、无固定流程、无必走路径、无写死台词。  
2. **只给行为倾向**：角色有反应基准；对话与行为由 AI 自由生成。  
3. **只放触发钩子**：事件 / 解锁 / 反转均为**条件触发**；触发权在玩家对话与行为。  
4. **世界自转**：全程无为 → 倒计时归零 → 宗门可覆灭；玩家任何对话、选择、表态均可正向/反向改写走向与结局。  
5. **一条线索 = 一个意图 = 一次事件包**（最硬契约）：  
   - 「解锁线索」= 天道发出一包与 `AdjudicationResult` **同构**的材料：`events` + `state_ops` / `belief_ops` / `world_flag_ops`（及必要 narrative）。  
   - **凡要落地的剧情变化**——开地图、坐实假信、查阵、动摇林溯、洛晴托付、通告、开战……**一律**走这条；禁止台词关键词旁路、禁止日历表旁路、禁止 Graph 写死分镜旁路。  
   - Mock 用 `simulate_clues` / `simulate_intents` **注入同构包**；LLM 天道直接产出同构字段。引擎**不扫台词念咒**。  
6. **硬状态是事件包落地后的结果**；见闻簿 / 案线 / 灰字 UI **只投影**已写入的 `flags` / `beliefs` / `world_flags`，不「另发一条进度」。  
7. **不写死平行数值轨**：不搞独立「全局信任值 / 人情点」养成。角色私有 flag（如洛晴 `trust_player`）可以有；`evidence_level` 仅调查副产品，非线索本体。  
8. **引擎边界**：内容在世界包种子 + 钩子 + 天道事件包；LLM 与 Mock 必须走同一落地路径。

## 1. 世界底层锚点（开局种子，非「禁改结局」）

> **锚点 = 仿真起点与信息差基准**，不是「玩家永远改不了」。  
> 玩家通过线索事件流，可以改写因果、立场、结局标签；只是不能靠念一句台词就绕过事件包直接改权威。

### 1.1 三层真相（信息差基准）

| 层 | 内容 |
|---|---|
| **公开共识** | 玄阴殿来函称秘境将开、全宗筹备仪式（对外话术；与劫数倒计时并行，**不**等于日历剧情表）。无掌门；表面由**沈监察**与**云烨**共治。 |
| **半公开异常** | 护山阵偶发异动；人心浮动、谣言；高层议事频而无结论。 |
| **核心隐秘** | ① 密信为**假**；② 护山阵阵眼已种**血阴咒**，开启仪式≈引爆；③ **林溯**为玄阴殿内应；④ 至宝**落霞剑髓**由落云子临终交**洛晴**藏匿；⑤ 落云子坐化实为**玄阴殿下毒**所致。 |

> ④⑤ 写入开局种子 flags / 信念，并严格信息差（见 §2）。向玩家/多数 NPC **露出**时，必须走对应线索事件包。

### 1.2 核心运转规则

**双轨时间**

- **自然日**：每日行动点有限；`max_days` 与劫数初始 **21** 对齐。  
- **劫数倒计时**：初始 **21**（`xuanyin_countdown`）；默认日终 -1；玩家行为经事件包可加速/减缓。

**信念传导**

- 玩家对某 NPC 说出的信息，按该 NPC **性格 / 立场 / 社交圈**进入传谣与信念。  
- 锚定 **RumorPass**（同地、延迟、跳数、失真）+ 天道/心智写入的 belief_ops。  
- **倾向扩散**，禁止「固定 N 天全图同步」。

**权限与解锁**

- 锁闭区域、涉密信息：**无固定解锁日**。  
- 触发只认：相关线索事件包已落地 → flags / 信念 / 通告 / 引荐等硬状态。  
- 禁止单独维护「人情点数」。

**世界自转**

- 无为：倒计时走完 → 仪式/引爆路径可致覆灭。  
- 有为：假信、血阴、内鬼、托付等均可被事件流改写（见结局标签）。

## 3. 人物 · 行为倾向（AI 基准）

每人只保留：动机、默认态、正/负向倾向、秘密钩子。  
**不限制**台词、出场顺序；**默认活动区**仅为倾向，开局聚拢见 §8.3。

### 3.1 云烨（大师兄）

- **动机**：守住落霞基业，查清异常根源。  
- **默认态**：寡言；庶务与宗主殿一带；与人保持距离；暗查「有内鬼」，不锁定具体人。  
- **正向**：玩家给有效线索、护宗立场清晰 → 渐信，可共享进展，愿为玩家**引荐/担保**进入执法相关、藏经许可等。  
- **负向**：帮林溯开脱、干扰调查 → 提防、限制活动、监视。  
- **钩子**：持有落云子去世当日异常记录；相关硬状态满足后可主动出示——由对话意图 → 天道事件包触发，非日程、非数值条。

### 3.2 林溯（二师兄）

- **动机**：借玄阴扶持争位，压过云烨。  
- **默认态**：广场、任务堂；广结善缘；推进仪式叙事；掩盖密信破绽。  
- **正向**：玩家有利用价值、站秘境说法 → 示好、差事、便利、「内部消息」。  
- **负向**：查密信、戳破绽 → 先安抚试探，再使绊（谣言、扣便利、黑手）。  
- **钩子**：不知血阴会炸宗；**不知**落云子为玄阴下毒；得知炸宗真意图后可**动摇**（`lin_su_*` 线索事件包）。权威：`allegiance=xuanyin`，可 `exposed`。

### 3.3 洛晴（小师妹）

- **动机**：守遗命、护落霞剑髓、查师父死因。  
- **默认态**：**倾向**后山静修；公共区**低频率**；开局须保留**偶遇可能**，禁止做成「整月只存在于传闻、玩家无处触发」。  
- **正向**：持续善意、不出卖、分享异常线索 → 回避→提醒→托付→联手。  
- **负向**：硬探至宝、站林溯、泄露其行踪 → 封闭、深藏。  
- **钩子**：剑髓 + 遗命；托付后可全盘（分阶段信念/flags，经事件包推进，非一键倾倒）。  
- **实现**：沿用 `trust_player` / `trusts_player` / arc flags，不另造全局信任条。

### 3.4 沈监察（执法长老）

- **动机**：法度；无实证不动，有实证雷霆。  
- **默认态**：执法堂；通告发声；疑密信但缺锤。  
- **正向**：合规实证、按程序报案 → 立案、通告、开放相关权限。  
- **负向**：无据造谣、违规 → 禁足、盘查、取消客卿便利。  
- **钩子**：弟子档案；立案后可调林溯异常出入。唯一 `can_proclaim` 权柄脸。通告 = 意图 → 事件包，不扫「通告」二字台词。

### 3.5 苏婉（三师姐）

- **动机**：正统秩序、程序正义。  
- **默认态**：巡礼；零容忍混乱；服沈之法，不服云烨独断。  
- **正向**：守规矩、助其维持秩序 → 程序助力、殿外相关便利。  
- **负向**：违规传谣 → 呵斥、上报。  
- **钩子**：秩序崩塌时可走「秩序反噬」倾向（结局标签已有接口）。

### 3.6 赵浦（任务堂）

- **动机**：机缘里捞利益；情报人情皆筹码。  
- **默认态**：任务堂；散播秘境话术；差事换人情。  
- **正/负**：给甜头↔断财路。  
- **钩子**：经手密信抄录；细问可见**格式漏洞**（假信线入口之一）。

### 3.7 明镜（藏经阁）

- **动机**：守典，认牌不认人。  
- **默认态**：藏经阁；无许可拒入。  
- **正/负**：正规许可/补残卷↔硬闯偷书。  
- **钩子**：旧录含血阴记载、剑髓来历、下毒旁证残片。

### 3.8 刘大壮（伙房）

- **动机**：安稳开饭（可保留「攒钱」等生活钩，不强制）。  
- **默认态**：伙房；八卦节点；听风是雨。  
- **正/负**：给甜头聊八卦↔得罪则抹黑。  
- **钩子**：见过林溯深夜往山门一带（传谣级，非实锤）。

### 3.9 韩铁（执法司事）

- **动机**：保职；可有把柄类生活压力（赌债等）作**可选钩子**，非必走勒索线。  
- **默认态**：随沈办事。  
- **钩子**：经手出入/笔录；可漏林溯记录（条件触发）。

### 3.10 岳山（山门）

- **动机**：守门；可有家人牵挂作可选钩。  
- **默认态**：认令盘查。  
- **钩子**：见过定期来寻林溯的外来人（传谣/目击级）。

### 3.11 白问舟（客司）

- **动机**：对接客卿，求稳求升。  
- **默认态**：客气、少涉阴谋。  
- **钩子**：客卿与外来登记；开局引导口（门径线索事件包）。

## 4. 三条主线 · 倾向与钩子（无顺序、非必走）

推进 = **线索事件包落地** → 信念 + flags + 事件卡。案线 UI 只投影，不是养成条。

### 4.1 假信线

**阶段倾向**：风闻 → 疑窦 → 坐实 → 公开  

| 跃迁 | 示例条件（经事件包落地） |
|---|---|
| → 疑窦 | 矛盾说法；密信格式/传讯路径异常（赵浦等）→ 写入调查 flags / 信念 |
| → 坐实 | 书证或人证 → `letter_exposed` 等（如线索 `letter_exposed`） |
| → 公开 | 沈通告意图事件包；或传谣仿真使大范围信念已不信秘境 |

### 4.2 血阴线

**阶段倾向**：异兆 → 查阵 → 破阵 / 爆发  

| 跃迁 | 示例条件（经事件包落地） |
|---|---|
| → 查阵 | 亲历异动 / 调查意图 → `investigating_curse` 等（如线索 `investigate_curse`）；**禁止**靠台词关键词「血阴」念咒 |
| → 破阵 | 阵图认知 + 解法 + 联手执行者（状态组合）。**不强制**动用落霞剑髓；剑髓主在人心线。洛晴同盟且持髓完成镇压 → 可选标签「剑髓安宗」 |
| → 爆发 | 倒计时归零；或仪式被强行启动 |

标志位：`blood_curse_*`、`sect_stabilized`、`crisis_*`；`jian_sui_required_for_disarm=False`。

### 4.3 人心线

**阶段倾向**：隔心 → 托重 → 同盟 / 反目  

| 跃迁 | 示例条件（经事件包落地） |
|---|---|
| → 托重 | 洛晴相关 flags 累积；无出卖/硬探至宝 |
| → 同盟 | 共享核心线索、愿共担风险（意图 → 如 `ally_shi_mei`） |
| → 反目 | 泄露剑髓；明确站林溯害她 |

至宝定名**落霞剑髓**、下毒真相进入托付后的信念——由事件包写入，非 UI 直接塞。

## 5. 动态钩子库（条件触发，非日程表）

禁止「第 N 天必刷事件」。下列均为**条件满足时可由内容钩子 / 天道抛出的框架**；对白自由，**结果落地必须成事件包**。

### 5.1 氛围向（高频、轻）

- 同地多人闲聊 → 可插话改局部信念，经传谣外溢。  
- 沈通告 → 大范围信念波动。  
- 伙房传谣 → 可纠正 / 加码 / 歪曲。

### 5.2 线索向（中频）

- 护山异动：证据类 flag 或倒计时偏低等条件。  
- 林溯拉拢 / 云烨试探 / 洛晴偶遇：条件钩子——**保证可触发**，非固定日程。

### 5.3 临界向（低频）

- **封山聚拢**：倒计时很低或假信半公开等 → 反向锁定、NPC 向核心区聚（§8.2）。  
- **内鬼揭发**：`exposed` 等实锤 → 立场分化。  
- **血阴爆发**：倒计时归零或仪式启动 → 终局结算。

## 6. 结局倾向（标签组合，非写死文本）

| 倾向 | 典型标签组合（示例） |
|---|---|
| 破局 | 假信已揭 + 血阴已解 + 内鬼已揭 → 宗门暂稳 |
| 残局 | 血阴暂镇 + 内鬼未除 + 核心伤亡 → 宗门危局 |
| 覆灭 | 血阴将爆/危机已燃 + 宗门溃散 → 落霞覆灭 |
| 人心收束 | 洛晴同盟 / 托付 / 得闻遗命 / 凶信 等 |
| **站队向（轻量）** | `站林溯` / `站云烨` / `独善其身` —— 只读硬状态打标签 |

### 6.1 站队标签判定

| 标签 | 条件（只读） |
|---|---|
| 站林溯 | 玩家 `faction_lean_lin` 且林溯仍在、未被 `exposed` |

| 独善其身 | 无上述站队；且未与洛晴同盟、假信未揭（低卷入） |
| 剑髓安宗 | 血阴已解 + 洛晴同盟 + 仍持剑髓（叙事加分，非破阵门槛） |

## 8. 开局开放、地图锁与线索展示（条件制，非日历）

### 8.1 开局体验

高自由 ≠ 开局空洞：

- 开局须能在**生活圈 + 广场**碰到足够主线脸。  
- 洛晴须存在「可触发的偶遇」。  
- 须知只给公开共识 + 行动方式，不剧透核心隐秘。  
- **禁止**「第 N 天自动开门 / 自动给线索」。

### 8.2 地图：起始开放集 + 条件解锁

| 集合 | 地点（建议） | 如何变化 |
|---|---|---|
| **开局即开放** | 外门客居、伙房、山门、落霞广场 | 起始状态 |
| **条件解锁** | 执法堂、任务堂、藏经、宗主殿、内门、后山、演武、长老院等 | 线索事件包落地（引荐/通告/flags）——**无固定天数** |
| **临界反向锁** | 封山时外门/山门等 | **`xuanyin_countdown ≤ 5`**，或 **`letter_exposed` 且 `countdown ≤ 10`** |

### 8.3 开局人物聚拢（默认落点倾向）

| 角色 | 开局倾向落点 |
|---|---|
| 玩家、白问舟 | 外门客居 |
| 刘大壮 | 伙房 |
| 岳山 | 山门 |
| 林溯、赵浦、云烨、苏婉 | 广场一带 |
| 沈、韩铁 | 执法堂（锁则靠偶过广场/通告感） |
| 洛晴 | 后山倾向 + **条件偶遇**公共区 |
| 明镜 | 藏经（锁内） |

日终移动仍由意图批次/倾向决定，不按日程赶人。

### 8.4 线索：事件包 + 投影（勿再写反）

1. **发线索** = 天道发出同构事件包（见 §0.5）。  
2. **硬状态** = 事件包 apply 之后的 flags / beliefs / world_flags。  
3. **UI**：  
   - 见闻簿：玩家信念列表（按 source / belief_id 约定分类，不扫命题词表）  
   - 案线：假信 / 血阴 / 人心——只亮已成立硬状态阶段
